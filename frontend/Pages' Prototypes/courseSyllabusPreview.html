Programación Orientada a Objetos

A-conferencia

Presencial

Bachillerato

ENGI-223

N/A

2 x 1.5 hrs por semana

3 créditos

Este es un curso para introducir a los estudiantes a la programacion orientada a objetos. En el mismo se estara utilizando JAVA como lenguaje de programacion. El enfoque del curso es mostrarle los conceptos claves de la programacion orientada a objetos, estos siendo la herencia, el polimorfismo, la encapsulacion, y la abstraccion.

A continuación iría la justificación del curso.

Al culminar el curso, el estudiante debe:

Objetivos del Curso CAEP
1. Comprender la encapsulación 1.1, 2.1, 3.1
2. Comprender la abstracción 1.2
3. Comprender el poliformismo 1.3
4. Comprender la herencia 1.4
5. Resolver problemas utilizando la programacion orientada a objetos. 1.5
6. Entender la diferencia entre datos privados, protegidos y publicos 1.6
